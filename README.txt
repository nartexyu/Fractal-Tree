README for Part 2 Homework (Fractal Tree)

Run index.html in any web browser (preferably Google Chrome). No downloads or compiling necessary.

Source code for calculating and drawing the fractal tree is in fractal.js. Recursive approaches (RC_approach) are named in the comments. Some are mutually recursive with each other.

Libraries used include bootstrap for styling and p5.js for drawing of rectangles and circles onto a html canvas.
